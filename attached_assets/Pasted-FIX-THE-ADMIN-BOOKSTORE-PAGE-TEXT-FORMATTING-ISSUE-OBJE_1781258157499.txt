FIX THE ADMIN BOOKSTORE PAGE TEXT FORMATTING ISSUE

OBJECTIVE

The Admin Bookstore page currently displays book information in an extremely narrow column, causing the title, author, category, pricing, CV, royalty information, and description to wrap vertically and appear stacked one word at a time.

The goal is to redesign the product card layout so that all text displays naturally from left-to-right in a professional horizontal format while remaining responsive on desktop, tablet, and mobile devices.

CURRENT ISSUE

The following fields are displaying incorrectly:

* Book Title
* Author Name
* Category
* Product Type
* Price
* CV
* Sales Count
* Royalty Percentage
* Platform Percentage
* Book Description

The content area is too narrow, causing text to wrap vertically.

DESIRED LAYOUT

Each book card should have three sections arranged horizontally:

LEFT COLUMN

* Book Cover Image

CENTER COLUMN

* Book Title
* Author
* Category
* Product Type
* Price
* CV
* Sales
* Royalty Information
* Description

RIGHT COLUMN

* Edit Button
* Buy Button
* Feature Button
* Best Seller Button
* Suspend Button
* Any future action buttons

VISUAL STRUCTURE

[ Book Cover ]
[ Book Information ]
[ Action Buttons ]

Everything should align horizontally across the card.

Locate the Admin Bookstore component.

Possible file locations:
